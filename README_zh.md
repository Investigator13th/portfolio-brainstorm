# 🎨 Portfolio Brainstorm | 个人主页构思助手

*用对话塑造你的个人主页，从构思到设计方向一氣呵成。*

通过自然对话引导你完成个人主页/作品集网站的构思过程，帮助你理清页面要传达的核心信息、目标受众以及最佳设计方向，然后再开始编码。

---

### ✨ 核心亮点

-   **💬 对话式引导**：一次只问一个问题，逐步帮你理清个人品牌定位与展示策略。
-   **🎯 以目标为导向**：聚焦于访问者在前 30 秒内应该了解什么，而非装饰性设计。
-   **🛠️ 多场景覆盖**：适用于求职主页、实习作品集、学术档案、开发者/设计师作品展示等。
-   **🧠 设计决策清晰**：从内容结构到视觉风格，帮你做出有充分理由的设计选择。

---

### 🚀 快速开始

想要为求职季规划个人主页，只需在对话框中输入：

> **"请启用 Portfolio Brainstorm 技能。我是一名 AI 产品方向的应届硕士生，正在准备秋招，需要一个个人主页来展示我的项目经历和博客，帮我规划主页结构。"**
