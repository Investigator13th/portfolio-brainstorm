# 🎨 Portfolio Brainstorm | 作品集头脑风暴

*在写下第一行代码前，帮您构思最完美的作品集网站。*

优秀的个人主页不仅是代码的堆砌，更是您个人品牌的最佳呈现。本技能通过循序渐进的自然对话，协助您明确受众、提炼核心内容并确立令人印象深刻的视觉方向。

---

### ✨ 核心亮点

-   **🎯 极致受众导向**：精准锁定您的访客（HR、潜在客户、行业大佬），从而定制最有效的信息架构。
-   **📑 核心内容梳理**：从繁杂的经历中抽丝剥茧，帮您筛选出最具含金量的项目与亮点。
-   **🖼️ 高级视觉定调**：无论是极简主义还是电影质感，为您探索最契合的视觉设计语言。
-   **🛣️ 渐进式轻松引导**：拒绝枯燥的长篇问卷，通过一次一个问题的高质量对话，轻松构建网站蓝图。

---

### 🚀 快速开始

准备好构思您的个人主页了吗？只需复制以下指令发给 AI：

> **"请启用 Portfolio Brainstorm 技能。我想做一个新的个人作品集网站，让我们开始头脑风暴吧！"**
