# portfolio-brainstorm

"Use this skill whenever the user wants to create, redesign, plan, or improve a personal homepage, portfolio website, resume site, internship/job-search landing page, academic profile site, or project showcase. This skill is especially relevant when the user says things like '个人主页怎么做', 'portfolio', 'personal website', '秋招/实习主页', '作品集网站', '帮我规划主页结构', or asks what content/style/sections their homepage should include. It guides a conversational, one-question-at-a-time discovery process and produces a clear homepage design direction before implementation."
