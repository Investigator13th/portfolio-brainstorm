# 🎨 Portfolio Brainstorm | 作品集头脑风暴

Help the user shape a personal homepage or portfolio site through natural dialogue. Focus on audience, content strategy, and visual direction before jumping into code.
通过自然语言对话，协助用户规划个人主页或作品集网站。在进入代码实现之前，深度聚焦于受众分析、内容策略及视觉方向的打磨。

---

### ✨ Features | 功能亮点

-   **🎯 Audience-Centric** | **受众导向**：明确您的访客是谁（HR、导师、客户），从而定制最有效的信息架构。
-   **📑 Content Inventory** | **内容梳理**：从教育经历到项目亮点，帮您筛选出最能体现价值的核心内容。
-   **🖼️ Visual Direction** | **视觉定调**：从极简主义到电影感风格，探索最契合您个人品牌的视觉语言。
-   **🛣️ Guided Process** | **渐进式引导**：拒绝漫长的问卷，通过一次一个问题的对话，轻松得出设计方案。
